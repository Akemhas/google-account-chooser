Redirect Behavior
These controls define when the extension asks you to choose an account and when it should quietly respect existing account hints.
If checked, links that already contain an explicit Google account hint such as authuser will open directly.
If checked, Google service links clicked on external websites will open through Account Chooser.
If checked, direct Google service visits such as typed URLs and bookmarks will also route through Account Chooser.
If checked, links clicked while already inside a supported Google service can also route through Account Chooser.
If checked, matching remembered account rules will add an authuser hint instead of forcing Account Chooser.
